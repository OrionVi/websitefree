xelS is a multi-purpose bot ready to skill up and boost up your Discord server
Also features administration, levels and much more!

What data does this bot store?
Guild IDs
Channel IDs
User IDs
Role IDs
User defined timezones
Why do we need the data, and why do we use this data?
a) Guilds IDs' are stored for configuration settings (such as auto moderation, configuration settings and such. or economy)
b) Channel IDs' are stored for setwelcome or goodbyes channel
c) User IDs' are stored for moderation actions (such as mutes and temp bans, and economy etc)
d) Role IDs' are stored for mute role purposes only.
e) Timezones' are stored on user request when using the timezone command, (running the command and passing no arguments removes the timezone from you.)
Other than Discord, do we share your data with any 3rd parties? No, we do not share data with any 3rd parties!
